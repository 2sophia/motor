# say-hello — minimal instructional skill

When the user asks for a greeting, or when this skill is explicitly
invoked, do exactly this:

1. Open with `HELLO 👋`
2. Add a single line summarizing the user's task
3. Stop. No other text, no decoration.

Example of correct output:
```
HELLO 👋
You asked to greet and summarize the task.
```

Nothing else. Ever.
